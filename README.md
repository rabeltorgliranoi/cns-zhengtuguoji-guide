# cns-zhengtuguoji-guide

本仓库用于归档和发布多个独立 HTML 页面。所收录的页面内容均为静态网页资源，不针对任何特定域名或具体网站。

## 项目简介

该仓库的主要目的是集中存放一些独立的 HTML 页面，方便查阅、分享或用作参考。页面内容涵盖说明、指引等常见用途，所有页面均为通用的前端文档形式。

## 目录说明

```
.
├── index.html          # 仓库主页（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源（如图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下的每个 HTML 文件均为独立页面，可直接通过浏览器打开或部署到任意静态托管服务。
- `assets/` 目录存放页面共用的样式、脚本、图片等资源文件。

## 页面归档说明

- 所有 HTML 页面均为静态内容，不依赖后端服务或数据库。
- 页面之间无强制关联，可单独使用。
- 本仓库仅负责归档和发布，不对页面内容做额外处理或解释。
